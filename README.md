# Trabalho-AED2
Trabalho da Disciplina de Algoritmos e estruturas de Dados 2, implementação de grafos em C
O trabalho tem como objetivo a criação de um grafo nao direcionado a partir de dados recebidos pelo usuario como quantidade de vertices e suas arestas respectivamente, além de apresentar para o usuario suas pontes, articulaçoes e o caminho minimo do primeiro vertice ao ultimo.
